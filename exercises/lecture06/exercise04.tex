\begin{Aufgabe}[Page Segmentation]{}
Implement the run-length smearing algorithm with all four steps:
\begin{itemize}
\item Horizontally smear the binary image obtained in the previous task.
\item Vertically smear the binary image obtained in the previous task.
\item Obtain an image by performing pixel-wise AND operation on the two smeared
  images.
\item Horizontally smear the obtained image to obtain the final bitmap.
\end{itemize}
\end{Aufgabe}
